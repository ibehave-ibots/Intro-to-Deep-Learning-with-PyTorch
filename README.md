**STEP 1: Clone this repository**

Alternative A) Clone via Source Control (the GUI) in VS Code. After clicking "Clone Repository" you paste the following link when prompted: https://github.com/ibehave-ibots/Intro-to-Deep-Learning-with-PyTorch.git

Then click "Open Folder"

Alternative B) Clone via the terminal. Paste the command below:

```
git clone https://github.com/ibehave-ibots/Intro-to-Deep-Learning-with-PyTorch.git
```

Then navigate to the folder using the following command:

```
cd Intro-to-Deep-Learning-with-PyTorch
```

**STEP 2: Install the environment**

In the terminal, use pixi to install the environment by pasting the following commands

```
pixi run install-kernel
pixi shell
```

or, you can use conda:
```
conda env create -f environment.yml
```
